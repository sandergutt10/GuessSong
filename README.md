# Guess Song (Android)

Android-приложение-игра, в котором пользователю необходимо угадать мелодию.
В проекте заранее заложен набор песен, реализована система подсказок и несколько уровней сложности.
Приложение разработано на Kotlin с использованием Jetpack Compose.

Технологии
- Kotlin
- Android SDK
- Jetpack Compose
- Material Theme
- Gradle

Описание приложения
Пользователю предлагается выбрать уровень сложности, после чего начинается игра.
На каждом шаге воспроизводится мелодия и отображаются варианты ответов.
Пользователь может использовать подсказки для упрощения выбора.
По завершении игры отображается экран победы или поражения.

Уровни сложности
- Лёгкий — небольшое количество вопросов
- Средний — среднее количество вопросов
- Сложный — увеличенное количество вопросов

Структура проекта

app/
- manifests/
  - AndroidManifest.xml

- kotlin+java/
  - com.example.guesssong/
    - model/
      - Song.kt          Модель песни
      - SongOption.kt    Варианты ответа
    - ui.theme/          Тема и стили приложения
    - view/
      - GuessMelodyApp.kt   Корневой composable приложения
      - GuessScreen.kt      Экран угадывания мелодии
      - LooserScreen.kt     Экран поражения
      - WinnerScreen.kt     Экран победы
      - MainMenu.kt         Главное меню и выбор сложности
    - MainActivity.kt       Главная Activity
    - Utils.kt              Вспомогательная логика и данные песен

- res/
  Ресурсы приложения

- Gradle Scripts
  Скрипты сборки Gradle

Архитектура
- UI реализован с использованием Jetpack Compose
- Навигация построена на composable-функциях
- Модели данных вынесены в отдельный пакет model
- Логика игры и подсказок реализована на уровне view и utils

Функциональность
- Предустановленный набор песен
- Три уровня сложности
- Подсказки во время игры
- Проверка ответов пользователя
- Экран победы
- Экран поражения
- Главное меню с выбором сложности

Запуск проекта
1. Открыть проект в Android Studio
2. Дождаться завершения Gradle Sync
3. Запустить приложение на эмуляторе или реальном устройстве

Требования
- Android Studio
- Android SDK
- Минимальная версия Android задаётся в настройках проекта
